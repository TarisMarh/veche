Проект Вече

Цель проект

Целью проекта являетсяц создание и поддержка площадки для общественного контроля над муниципальными структурами управления. На первом этапе предполагается создание возможности для отслеживания общественных слущаний, информация о которых усиленно закапывается в глубины сайта городской администрации вместо вывешивания на главной странице или на специальной странице, на которую ведёт ссылка с главной странице. Собственно, функцию это специальной страницы я и попытаюсь реализовать.

2. Задачи

  1. Публикация тем общественных слушаний, полной сопутствующей информации (зеркалирование на сайт документов, аудио+ и видеозаписей), публикация новостей по слушаниям.

  2. При изменении документов должна сохраняться история изменений, чтобы можно было отслеживать телодвижения заинтересованных сторон.

  3. Обеспечение возможности общения по заявленным слушаниям.

  4. Обеспечение уведомления о новых собитиях.

  5. Обеспечение срочных уведомления в случае неожиданных изменений.

  6. Предоставлять возможность голосования по проблеме. Желательно даже, чтобы оно имело какую-то юридически признанную форму, принимаемую к рассмотрению. То есть голосование поимённое, никаких безликих студентов, никаких выдуманных персонажей, только настоящие граждане, проживающие в городе. Город должен знать своих героев.
